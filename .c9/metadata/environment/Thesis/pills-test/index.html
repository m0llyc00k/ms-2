What's this

","\t\t

","\t\tBox2DJS is a JavaScript port of Box2D Physics Engine.","\t\tTo tell the truth, this is converted from Box2DFlashAS3_1.4.3.1 in an automatic manner.","\t\t(The reason why not Box2DFlashAS3_2.0.0 based is simply because I overlooked the renewal.)","\t\t

How to use

","\t\t

","\t\tBecause this libray does not have a lazy-loading system now, you should load all classes before starting your simulation.","\t\tTo make things worse, each library has a bit complecated dependency each other so that loading libraries in wrong order may cause a fatal error. ","\t\tTo avoid such a trouble, it is strongly recomended to copy the header part of this file or `index.html' including the downloaded zip file.","\t\t

","\t\t

","\t\tConcerning the Box2D APIs of the library, they are completely same as Box2DFlashAS3. Please google information about it.","\t\t
